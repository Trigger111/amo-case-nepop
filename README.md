# AMO Case — аудит цінового рішення 6.99 → 9.99

**Автор:** Олександр Непоп · Genesis Analytics Camp 3.0

Команда АМО, вітаю! 
Дякую Вам за створення такого цікавого та нетривіального кейсу. Я прожив з ним неділю, проганяв різні гіпотези, звіряв цифри, сумнівався, але врешті решт готовий презентувати Вам свою роботу. Сподіваюся Вам буде комфортно працювати з кодом та презентацією, внизу залишив roadmap та невеличку інструкцію по запуску. Сподіваюся моя аналітика правильна та допоможе бізнесу ухвалити правильні рішення, проаналізувати свої помилки.

Пів року тому компанія підняла ціну тижневої підписки 6.99 → 9.99 (+42.9%) на основі
A/B-тесту, який показав «конверсія не впала». CEO не бачить цих відсотків у фактичному
Revenue. Цей проєкт відповідає на 4 питання кейсу: конверсія, LTV, Revenue за
26.01–16.03, ROAS — і дає фінальну рекомендацію щодо цін.
